# Web Quiz (HTML & CSS)

Quiz responsif sederhana yang dibuat hanya menggunakan HTML dan CSS. Tidak menggunakan JavaScript — jawaban benar ditampilkan menggunakan elemen `<details>` sehingga pengguna dapat memilih sendiri dan memeriksa kunci.

Cara menjalankan:

1. Buka file `projects/quiz/index.html` di browser.
2. Halaman ini responsif dan dapat dibuka dari perangkat mobile maupun desktop.

Catatan:

- Tidak ada penyimpanan skor atau interaksi dinamis tanpa JavaScript.
- Silakan tambahkan gambar atau modifikasi soal sesuai kebutuhan.
