# Shortest-Path (Python / Flask)

Project ini menggunakan algoritma Dijkstra dan disajikan dengan backend Flask serta halaman HTML. Kode sumber utama berada di repository GitHub berikut:

- https://github.com/AdityaIyan/Shortest-Path-Dijkstra.git

Cara menjalankan (clone dan jalankan):

```bash
git clone https://github.com/AdityaIyan/Shortest-Path-Dijkstra.git
cd Shortest-Path-Dijkstra
# Sesuaikan instruksi di README repo tersebut; kemungkinan ada requirements.txt
pip install -r requirements.txt
python app.py
```

Catatan: projek ini dihubungkan dari projek utama sebagai sub-proyek eksternal — tombol GitHub pada UI akan membuka repository ini.
