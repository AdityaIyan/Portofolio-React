# Project Images

Letakkan gambar project Anda di folder ini dengan nama file berikut:

- **rsbp-wound.jpg** - Gambar untuk "RSBP Wound Chatbot"
- **tokokomoto.jpg** - Gambar untuk "TOKOKOMOTO E-Commerce"

## Format Rekomendasi

- Format: JPG atau PNG
- Ukuran: 800x600px atau lebih besar (aspect ratio bisa fleksibel)
- Ukuran file: Lebih kecil dari 1MB untuk performa optimal

## Catatan

Setelah menambahkan gambar ke folder ini, pastikan nama file sesuai dengan yang tersebut di atas agar komponen Projects.jsx dapat menampilkannya dengan benar.
